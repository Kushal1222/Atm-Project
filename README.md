# Atm-Project
# 🏧 ATM Simulation Project

## 📌 Description
This is a simple ATM Simulation project built using Python.  
It allows users to perform basic banking operations like deposit, withdrawal, and viewing transactions.

## 🚀 Features
- PIN-based login system
- Check balance
- Deposit money
- Withdraw money
- View full statement
- Mini statement
- Change PIN

## 🛠️ Technologies Used
- Python
- Functions, Lists, Dictionaries
- datetime and time modules
